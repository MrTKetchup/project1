About me

History

Since graduating I have had a range of different careers varying from Nuclear Safety Consultant to Sous Chef to most recently as a Brewer at a small independent craft brewery located in Bath called Electric Bear Brewing Co.

I am now in the next chapter of my life, learning how to develop code and I have recently completed a 16-week Full Stack Track Bootcamp with iO Academy in Bath.

Portfolio

Have a look below at some of the projects I have been working on

BrickBreaker Game

This is my second project using Java. Initially I followed this tutorial by Awais Mirza to help me navigate some of the features of Java.
I intend on adding new features to mess with the physics of the game such as having the ball accelerate on contact with the paddle, difficulty settings and a more randomised collision handling.

Number Guessing Game

This is my first very small project using Java.
It's a very simple number guessing game. The user picks a difficulty and then has a number of tries to try and guess the correct number with a little bit of higher/lower hints to help get there.
To give it a whirl, click the download link above and type the following into CMD from the file's directory:
java -jar NumberGuessingGame.jar

Project Management API

For this project, as part of a team at iO Academy, I helped to design an Object-Oriented PHP API to work with a pre-built React front end so that projects were displayed detailing the people and tasks assigned to each with the correct success or error response message being displayed in line with a specified API documentation

Email Client Application

As part of a team at iO Academy, I helped to create an email client following a specified design which allowed the user to compose and display emails and signify whether they had been read/unread. This combined and improved my HTML, CSS and JavaScript proficiency through usage of the React and Bootstrap Frameworks.

Where in the World

In this project I have combined my JavaScript, HTML and CSS knowledge to create a application that interacts with https://restcountries.com/ API to display a range of countries and some facts about them.
I have implemented the Bootstrap Framework and also added the ability to search for a specific country by name with the results to update live as you type.

Caterpillar Countdown

As part of a team at iO Academy, we created a simple counting game that focused on using JavaScript. I was responsible for handling the click order logic so that the number cards would only highlight correctly if pressed in sequence, resetting them if they were pressed incorrectly.

Password Generator

Simple password generator app utilising JavaScript, HTML & CSS.
Logic created to randomly generate passwords to include uppercase & lowercase letters as well as numbers and symbols.
